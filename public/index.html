Implementación build
  1. Enlazar las librerías de js mediante la etiqueta script al html
  2. En el contenedor de las imágenes colocar una clase llamada container
  3. Todas las imágenes deben de tener su atributo alt
  4. Por último, llamar al plugin en el archivo js
